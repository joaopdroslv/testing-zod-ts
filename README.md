# Zod Testing Playground

This repository is a small collection of TypeScript files created to learn, explore, and experiment with Zod, a TypeScript-first schema validation library.
Each file focuses on a specific Zod feature or pattern, ranging from basic validation to more advanced use cases like refinements, transformations, unions, async validation, and recursive schemas.
